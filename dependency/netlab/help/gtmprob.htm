gtmprob

Purpose

Probability for data under a GTM.

Synopsis 

prob = gtmprob(net, data)

Description

prob = gtmprob(net, data) takes a GTM structure net, and computes the probability of each point in the dataset data.

See Also

gtm, gtmem, gtmlmean, gtmlmode, gtmpost
