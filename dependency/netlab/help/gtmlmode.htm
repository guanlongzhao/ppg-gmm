gtmlmode

Purpose

Mode responsibility for data in a GTM.

Synopsis 

modes = gtmlmode(net, data)

Description

modes = gtmlmode(net, data) takes a GTM structure net, and computes the modes of the responsibility distributions for each data point in data. These will always lie at one of the latent space sample points net.X.

See Also

gtm, gtmpost, gtmlmean
