demgtm2

Purpose

Demonstrate GTM for visualisation.

Synopsis 

demgtm2

Description

This script demonstrates the use of a GTM with a two-dimensional latent space to visualise data in a higher dimensional space. This is done through the use of the mean responsibility and magnification factors.
