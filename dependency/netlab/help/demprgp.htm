demprgp

Purpose

Demonstrate sampling from a Gaussian Process prior.

Synopsis 

demprgp

Description

This function plots the functions represented by a Gaussian Process model. The hyperparameter values can be adjusted on a linear scale using the sliders (though the exponential of the parameters is used in the covariance function), or by typing values into the text boxes and pressing the return key. Both types of covariance function are supported. An extra function specific parameter is needed for the rational quadratic function.
