gtmlmean

Purpose

Mean responsibility for data in a GTM.

Synopsis 

means = gtmlmean(net, data)

Description

means = gtmlmean(net, data) takes a GTM structure net, and computes the means of the responsibility distributions for each data point in data.

See Also

gtm, gtmpost, gtmlmode
