glmderiv

Purpose

Evaluate derivatives of GLM outputs with respect to weights.

Synopsis 


g = glmderiv(net, x)

Description

g = glmderiv(net, x) takes a network data structure net and a matrix of input vectors x and returns a three-index matrix mat{g} whose i, j, k element contains the derivative of network output k with respect to weight or bias parameter j for input pattern i. The ordering of the weight and bias parameters is defined by glmunpak.

See also 

glm, glmunpak, glmgrad
