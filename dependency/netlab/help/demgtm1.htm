demgtm1

Purpose

Demonstrate EM for GTM.

Synopsis 

demgtm1

Description

This script demonstrates the use of the EM algorithm to fit a one-dimensional GTM to a two-dimensional set of data using maximum likelihood. The location and spread of the Gaussian kernels in the data space is shown during training.
