eigdec

Purpose

Sorted eigendecomposition

Synopsis 

evals = eigdec(x, N)
[evals, evec] = eigdec(x, N)

Description

evals = eigdec(x, N computes the largest N eigenvalues of the matrix x in descending order. [evals, evec] = eigdec(x, N) also computes the corresponding eigenvectors.
