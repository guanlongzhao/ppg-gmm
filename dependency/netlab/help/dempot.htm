dempot

Purpose

Computes the negative log likelihood for a mixture model.

Synopsis 

e = dempot(x, mix)

Description

This function computes the negative log of the unconditional data density p(x) for a Gaussian mixture model. The data structure mix defines the mixture model, while the matrix x contains the data vectors.
