gtmmag

Purpose

Magnification factors for a GTM

Synopsis 

mags = gtmmag(net, latentdata)

Description

mags = gtmmag(net, latentdata) takes a GTM structure net, and computes the magnification factors for each point the latent space contained in latentdata.

See Also

gtm, gtmpost, gtmlmean
