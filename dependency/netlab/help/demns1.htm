Purpose

Demonstrate Neuroscale for visualisation.

Synopsis 

demns1

Description

This script demonstrates the use of the Neuroscale algorithm for topographic projection and visualisation. A data sample is generated from a mixture of two Gaussians in 4d space, and an RBF is trained with the stress error function to project the data into 2d. The training data and a test sample are both plotted in this projection.
