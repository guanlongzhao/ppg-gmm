glmfwd

Purpose

Forward propagation through generalized linear model.

Synopsis 

y = glmfwd(net, x)
[y, a] = glmfwd(net, x)

Description

y = glmfwd(net, x) takes a generalized linear model data structure net together with a matrix x of input vectors, and forward propagates the inputs through the network to generate a matrix y of output vectors. Each row of x corresponds to one input vector and each row of y corresponds to one output vector.

[y, a] = glmfwd(net, x) also returns a matrix a giving the summed inputs to each output unit, where each row corresponds to one pattern.
